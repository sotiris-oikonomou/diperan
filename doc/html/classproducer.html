VIRTUAL void producer::print_msg ( )
inline

Print Message

Prints hello world to the console. The function is marked "VIRTUAL" instead of "virtual" because the this class is not intended to be subclassed, but "virtual" is required for unit testing. Therefore, this class is compiled twice:

  • the main application defines VIRTUAL=
  • the unit test recompiles this source defining VIRTUAL=virtual
